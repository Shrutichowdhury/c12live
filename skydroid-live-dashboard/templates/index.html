How to enable real gimbal control
  1. Make sure your computer is on the same network as the camera (192.168.144.108).
  2. Click Test Connection to verify the control port is reachable.
  3. If the test passes, click Enable Real Control — gimbal commands will go directly to the camera.
  4. If the test fails, the control port or protocol may differ. Video streaming uses RTSP (ports 554/555) which is separate from the control channel.

The exact Ethernet control protocol for the C12 is not publicly documented. If Enable Real Control fails, commands will automatically fall back to Mock Mode.
